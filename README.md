# ai-craftkit

`ai-craftkit` is a personal toolkit for reusable AI-assisted development workflows.

It collects practical materials that help humans and coding agents work more effectively inside software repositories, such as:

- agent skills
- prompt files
- documentation templates
- repository inspection workflows
- architecture documentation patterns
- development and operations guidance for AI-assisted coding

The first focus of this repository is the `archdoc` skill, which generates agent-friendly repository documentation:

docs/REPO_MAP.md
docs/ARCHITECTURE.md
docs/OPERATIONS.md

These docs are intended to help a human or agent quickly understand what a repository contains, how it is structured, how it runs, and what should be checked before making changes.

## Why this name?

The name `ai-craftkit` is intentional.

It describes a small craft kit for AI-assisted software work: reusable tools, templates, and practices that can be copied into different projects.

I also chose this name to avoid clashes or confusion with projects named `ai-toolkit`.
